# Amani API REST

## Descripción

API REST del proyecto Amani, desarrollada con **Spring Boot 4.0** y **Java 21**.

## Herramientas de calidad

Este informe incluye los resultados de las siguientes herramientas de análisis:

| Herramienta | Propósito                              |
|-------------|----------------------------------------|
| PMD         | Detección de code smells y malas prácticas |
| Checkstyle  | Verificación de convenciones de estilo     |
| SpotBugs    | Detección de bugs potenciales              |
| JaCoCo      | Cobertura de tests                         |

## Tecnologías

- Java 21
- Spring Boot 4.0.3
- Spring Security + JWT
- Spring Data JPA + PostgreSQL
- Firebase Admin SDK
- WebSocket
- OpenAPI / Swagger UI
