# Dashboard de cobertura

El dashboard interactivo incluye KPIs con deltas, gráficos de tendencia, tablas filtrables de suites de tests y cobertura por clase (P0/P1/P2).

!!! tip "Ver dashboard interactivo"
    Abre [`coverage-dashboard.html`](coverage-dashboard.html) en tu navegador.

    ```bash
    # Linux
    xdg-open docs/reports/coverage-dashboard.html

    # macOS
    open docs/reports/coverage-dashboard.html
    ```
