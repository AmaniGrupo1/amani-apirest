CategoriaTicketSoporte

enum CategoriaTicketSoporte(source)

Enumeracion de las categorias posibles para un ticket de soporte.

  • tecnico - problemas tecnicos, errores o bugs.
  • cuenta - gestion de perfil, datos personales o acceso.
  • pago - consultas relacionadas con pagos y facturacion.
  • app - funcionamiento de la aplicacion movil.
  • otro - cualquier otra categoria no contemplada.
