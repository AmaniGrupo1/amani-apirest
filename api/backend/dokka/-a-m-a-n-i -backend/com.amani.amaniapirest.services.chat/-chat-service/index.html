ChatService

@Service
open class ChatService(source)

Servicio de dominio para la funcionalidad de chat.

Orquesta la comunicación entre PostgreSQL (canónico) y Firebase RTDB (tiempo real). La publicación a RTDB se realiza de forma única a través del evento de dominio MensajeNuevoEvent, procesado por MensajeEventListener.

Utiliza caché Caffeine (chatMessages) para reducir lecturas repetitivas de historial reciente. La caché se invalida automáticamente al enviar un nuevo mensaje.
