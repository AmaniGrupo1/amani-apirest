Usuario

@Entity
open class Usuario(source)

Entidad central de identidad y autenticación de la plataforma.

Concentra los datos comunes a todos los perfiles del sistema (paciente, psicólogo y administrador): credenciales de acceso, rol, estado de la cuenta y token FCM para notificaciones push. Cada Paciente o Psicologo se vincula a un Usuario mediante una relación 1:1.
