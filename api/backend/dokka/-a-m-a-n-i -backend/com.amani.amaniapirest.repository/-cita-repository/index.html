CitaRepository

interface CitaRepository : JpaRepository<T, ID> (source)

Repositorio de acceso a datos para la gestión de citas entre pacientes y psicólogos.

Centraliza las consultas relacionadas con el ciclo de vida de una cita: búsqueda por participante, filtrado por rango temporal, filtrado por estado y carga completa con todas sus relaciones para vistas de historial administrativo. Extiende JpaRepository para proveer operaciones CRUD estándar.
