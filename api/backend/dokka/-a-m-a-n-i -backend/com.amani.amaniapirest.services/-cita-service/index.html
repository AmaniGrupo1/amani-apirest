CitaService

@Service(value = "citaServiceGeneral")
open class CitaService(source)

Servicio central de gestion de citas.

Agrupa la logica de negocio para tres perfiles de usuario:

  • General (paciente) — CRUD completo de citas.
  • Administrador — CRUD con respuesta enriquecida (CitaAdminResponseDTO).
  • Psicologo — Consulta y cambio de estado de citas propias.

Publica eventos de dominio (CitaCreadaEvent, CitaCanceladaEvent) para desacoplar las notificaciones del flujo principal.
