CitaService

@Service
open class CitaService(source)

Gestiona las citas del paciente autenticado desde su perspectiva de uso.

Expone operaciones de consulta de citas propias (por identificador, por mes) y la cancelación de citas pendientes. La resolución de la identidad del paciente se realiza a partir del contexto de seguridad JWT mediante obtenerIdPacienteDesdeAuth.
