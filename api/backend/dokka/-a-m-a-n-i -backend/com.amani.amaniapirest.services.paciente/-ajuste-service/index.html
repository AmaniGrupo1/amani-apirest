AjusteService

@Service
open class AjusteService(source)

Gestiona las preferencias personales de configuración de cada usuario de la plataforma.

Cubre el ciclo de vida completo de los ajustes: idioma de la interfaz, activación de notificaciones, tema visual (oscuro/claro) y zona horaria. Si el usuario aún no tiene ajustes creados, los métodos de actualización parcial los generan con valores por defecto antes de aplicar el cambio solicitado.
