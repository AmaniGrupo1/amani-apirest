Controlador REST para operaciones de pago de pacientes.

Todas las rutas bajo /api/payments requieren autenticación JWT. El backend crea el PaymentIntent en Stripe y devuelve únicamente el clientSecret al cliente, nunca la clave secreta de Stripe.
