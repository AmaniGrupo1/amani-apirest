FirebaseConfig

@Configuration
@ConditionalOnProperty(name = "firebase.enabled", havingValue = "true", matchIfMissing = false)
open class FirebaseConfig(source)

Configuración del cliente Firebase Admin SDK.

Solo se activa cuando firebase.enabled=true en la configuración. Cuando la propiedad está ausente o es false, ninguno de los beans Firebase se registra y la aplicación arranca normalmente sin Firebase.

Credenciales (orden de resolución)

  1. Contenido del secreto inyectado por Spring Cloud GCP (property firebase-service-account).
  2. Ruta externa configurada en firebase.credentials-path.
  3. Classpath: serviceAccountKey.json.
  4. Application Default Credentials (ADC) automático en GCP.

Emuladores

Cuando firebase.emulator.enabled=true, se configuran las variables de entorno para conectar a los emuladores locales de Firebase Auth y RTDB.
