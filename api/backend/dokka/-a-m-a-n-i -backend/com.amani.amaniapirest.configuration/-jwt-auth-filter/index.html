JwtAuthFilter

@Component
open class JwtAuthFilter : OncePerRequestFilter(source)

Filtro JWT que se ejecuta una vez por petición HTTP.

Extrae el token del encabezado Authorization: Bearer <token>, lo valida y, si es correcto, establece la autenticación en el SecurityContextHolder para el resto de la cadena de filtros.
